## MODIFIED Requirements

### Requirement: 로그인 계정 표시
네비게이션 바는 로그인한 사용자의 계정 정보를 화면 크기에 따라 다르게 표시 SHALL.
- 모바일(md 미만): 이메일의 `@` 앞 아이디 부분만 표시 SHALL
- PC(md 이상): 이메일 전체를 표시 SHALL
두 경우 모두 녹색 점과 함께 배지 형태로 표시 SHALL.

#### Scenario: 모바일에서 아이디 표시
- **WHEN** 사용자가 모바일(md 미만) 화면에서 로그인 상태로 앱을 열면
- **THEN** 네비게이션 바에 이메일의 `@` 앞 아이디 부분이 배지로 표시되어야 SHALL

#### Scenario: PC에서 전체 이메일 표시
- **WHEN** 사용자가 PC(md 이상) 화면에서 로그인 상태로 앱을 열면
- **THEN** 네비게이션 바에 이메일 전체가 배지로 표시되어야 SHALL
