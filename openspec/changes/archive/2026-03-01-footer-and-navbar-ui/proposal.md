## Why

앱 출시를 앞두고 PC·모바일 양쪽에서 네비게이션 바와 하단 푸터의 가독성·레이아웃이 미흡하다. 모바일에서 로그인 계정 정보가 전혀 표시되지 않고, 푸터의 사업자 정보가 10px 폰트와 한 줄 파이프 구분 방식으로 읽기 어렵다.

## What Changes

- **네비게이션 바**: 모바일에서 이메일 `@` 앞 아이디 부분만 표시, PC에서는 전체 이메일 표시
- **푸터 폰트 크기**: `text-[10px]` → PC `text-xs`(12px), 모바일 `text-[11px]`로 가독성 향상
- **푸터 레이아웃 — PC**: 사업자 정보를 2줄 파이프 구분 유지, 세션 배지 우측 유지
- **푸터 레이아웃 — 모바일**: 사업자 정보를 항목별 줄바꿈으로 분리, 세션 배지 숨김(`hidden md:block`)

## Capabilities

### New Capabilities

없음

### Modified Capabilities

- `navbar-user-display`: 모바일에서 이메일 전체 숨김 → 아이디(@앞부분)만 표시로 요구사항 변경
- `footer-layout`: 푸터 사업자 정보의 반응형 레이아웃 및 폰트 크기 요구사항 변경

## Impact

- `App.tsx` — Navbar 컴포넌트(이메일 표시 영역), AppContent 컴포넌트(footer)
- 변경 파일 1개, 신규 파일 없음
- 기능 변경 없음, UI/레이아웃만 수정
